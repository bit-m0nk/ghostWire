Features

Everything you'd pay $10/mo for

Commercial VPNs charge monthly for what GhostWire gives you permanently on hardware you own.

🔐
IKEv2 / IPSec
Industry-standard strongSwan VPN. Native clients on every platform — iOS, Android, Windows, macOS, Linux. No app needed.
🛡️
DNS Ad Blocking
Block ads, trackers, and malware at the DNS level for every connected device. Per-user toggles. Custom whitelist and blocklist.
📊
Analytics + GraphQL
See what's being blocked, when, and by whom. GraphQL endpoint for flexible custom queries. Daily rollup summaries.
🌍
Multi-Server Fleet
Manage multiple VPN nodes from a single dashboard. Health monitoring, CPU/RAM graphs, latency tracking.
🤖
Bot Notifications
Get Telegram, Discord, or Slack alerts for connections, failures, and node events. One toggle — all platforms.
🔑
Two-Factor Auth
TOTP authenticator apps and email OTP. Admin force-reset for lost phone recovery. Per-user 2FA status in the admin table.
🧩
Plugin System
Install community plugins via zip upload. Plugins can add API routes, custom dashboard pages, and scheduled tasks.
🎨
Theming Engine
5 built-in themes — Indigo, Midnight, Forest, Rose, Amber. Build custom themes with a live-preview color editor.
